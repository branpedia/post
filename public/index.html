API Integration

Gunakan endpoint berikut untuk mengambil screenshot secara terprogram:

POST /api/screenshot

Parameter (JSON body):

  • url: URL website yang akan di-screenshot (wajib)
  • device: Jenis perangkat (desktop, tablet, phone) - opsional, default: desktop

Contoh:

{
  "url": "https://github.com",
  "device": "phone"
}

Kolaborasi Real-time

Fitur yang tersedia dalam mode multiplayer:

  • Berbagi screenshot ke semua anggota room
  • Chat real-time dengan anggota room menggunakan IP public
  • Melihat aktivitas pengguna lain berdasarkan IP
  • Mendownload screenshot yang dibagikan

Tips Kolaborasi

Setiap pengguna diidentifikasi oleh IP public mereka. Gunakan ID room yang unik untuk tim Anda.
